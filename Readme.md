# AI Academic Tutor using Ollama and LangChain

## Objective

This project is a console-based AI academic tutor that helps students learn, revise, generate quizzes, solve problems step by step, and create study plans using a locally hosted LLM.

## Technologies Used

- Python
- LangChain
- Ollama
- Llama 3.2:3B
- SymPy

## Why Ollama?

Ollama runs the LLM locally on the user's system. Therefore, no OpenAI API key is required.

## Features

1. LLM integration using Ollama
2. Zero-shot prompting
3. Few-shot quiz generation
4. Step-by-step reasoning
5. Role-based tutor
6. Reusable prompt templates
7. LangChain chains
8. Memory system
9. Smart agent with tool routing
10. Final console application

## Tools

- Calculator
- AI Study Planner
- Summarizer
